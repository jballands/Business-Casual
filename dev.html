Let's be real: nothing beats a custom-made website built from the ground up. There's nothing really as difficult, either. But you're better than frameworks! What are you to do?

Business Casual is a framework I originally built for my personal website, and now I'm sharing what I've learned with you. See what makes Business Casual tick and extend it with your own features, or just take bits and pieces to work with your own code. I promise I won't mind.

Why reinvent the wheel? Stay awesome.

Getting Started

Introduction to BC LESS

All of the static features of Business Casual are handled by files located in the less/ directory. These features include:

  • Flexors
  • Standard navigation bars
  • Images
  • Tables
  • Wells
  • Buttons
  • Customization

All files in the less/ directory are compiled with Grunt and Node. If you need help setting up Grunt, see the Getting Started section.

Heads up!
This section assumes you know how to write LESS code. It is not recommended that you go on, otherwise. Go to the LESS website to learn more.

Basics

Introduction to BC Javascript

All of the dynamic features of Business Casual are handled by a file called business-casual.js. These features include:

  • Sticky/magnetic navigation bars
  • Heroes
  • Glassboxes
  • Parallax

business-casual.js is minimized with Grunt and Node. If you need help setting up Grunt, see the Getting Started section.

Heads up!
This section assumes you understand and are comfortable with Javascript code. It is not recommended that you go on, otherwise.
